# Credits

Third‑party assets used in The Institute.

## 3D models

- **Spartan** by IronRus, purchased on Fab (Epic) under the Fab Standard Licence. Used as the animated 3D foot‑soldier model. The Standard Licence permits use inside this game but not redistribution of the raw asset files, so this repository must stay private.

## Animations

- Character animations (Sword And Shield: Idle, Walk, Slash, Death) from **Adobe Mixamo**, applied to the Spartan mesh. Free for use in projects under Adobe's Mixamo terms.

## Notes

- The final in‑game model (`assets/models/spartan-lite.glb`) combines the Spartan mesh with the Mixamo animations and re‑applied textures. Keep the repository private per the Fab Standard Licence.
